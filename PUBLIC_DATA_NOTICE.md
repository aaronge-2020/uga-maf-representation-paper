# Public Data Notice

This public repository includes only the curated inputs selected for redistribution and reproduction of the manuscript figures and tables.
The bundle excludes obsolete archives, scratch outputs, duplicate work directories, and inputs whose redistribution status was not needed for the current manuscript workflow.

See `DATA_MANIFEST.csv` for checksums and source categories for every included file.
See `DATA_NOT_INCLUDED.md` for intentionally excluded large or obsolete files.
