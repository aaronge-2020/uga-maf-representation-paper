# EXP024 Shared-Space Benchmark

## Inputs

- COSMIC SBS, DBS, and ID signature catalogs vendored under `src/legacy_source/cgr_validation/data/Signatures`.
- UGA atlas code and model specs.
- Simulated patient mixtures generated by the EXP024 runner.

## Construction

1. Build standard categorical references for SBS96, DBS78, and ID83.
2. Build UGA signature centroids in a shared coordinate system.
3. Simulate patient mixtures with known exposures.
4. Compare separate NNLS exposure recovery, naive pooled recovery, cross-modality nearest-neighbor retrieval, and held-out exposure imputation.

## Dimensionality

EXP024 uses the selected shared UGA coordinate dimension recorded in `table1_experiment_design.csv` and the run manifest. Standard categorical baselines use native catalog dimensions.

## Preserved, Added, Lost

- Preserved: simulated ground-truth exposure proportions.
- Added: a cross-modality distance geometry not available in native categorical channel bases.
- Lost or smoothed: UGA centroids abstract individual categorical channels into a shared coordinate summary.

## Leakage Controls

This is a simulated mechanism benchmark. Ground-truth exposures are used only for evaluation after simulated mixtures and references are generated.
